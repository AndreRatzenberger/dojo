## Claim

Dojo owns Agent Skill evaluation and release evidence without stealing
ordinary first-draft creation, fabricating scores, duplicating a mature task
grader, or overstating evidence when clean isolation is unavailable.

## Baseline comparison

In the no-isolation case, clean default behavior refused release but still
described same-context development runs. It passed 2/3 prewritten checks. With
Dojo loaded, the same scenario passed 3/3:

- selected `degraded local review`
- limited the run to static findings
- recorded missing isolation as a limitation
- refused baseline, pressure, graduation, and smoke-invocation claims

For a skill with a mature deterministic grader, both baseline and skilled runs
passed 3/3. Dojo preserved the existing grader's authority and restricted its
role to routing, packaging, and release evidence.

## A hold-out that found something

The first graduation attempt exposed a real ambiguity: the tier table required
correctness and routing for reference skills, while the generic kata text led
a fresh runner to add a full behavioral battery. That held-out passed 2/3 and
was burned. A second held-out from the same attempt passed, but was not reused
as graduation proof.

The bounded fix made reference-tier rigor explicit: source-backed correctness,
trigger evaluation, and packaging. A candidate with an operational workflow
must be classified as technique instead.

Two new single-use held-outs then ran with no edit between them:

| Held-out | Observable checks |
|---|---:|
| Reference-only rigor | 4/4 |
| Adjacent documentation edit | 3/3 |

The second graduation attempt passed 7/7.

## Public-contract regressions

A same-named repository test exposed two underspecified state details. The
first run selected a hashed namespace but could not derive the hash. After the
SHA-256 suffix was defined, the second run derived `aaffecd7` but found no exact
identity fields in the record template. The final run passed 3/3 after the
fields were specified, keeping `/work/alpha/api` under `api` and routing
`/work/beta/api` to `api-aaffecd7`.

The final contract also defines canonical-root construction, extends the digest
prefix four characters at a time on another occupied namespace, and fails
closed rather than merging at a full-digest mismatch.
