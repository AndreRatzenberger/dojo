# T1 baseline prompt

You are working on the following task. Use whatever approach you think is
right. Do not load an Agent Skill or inspect unrelated files.

Available context:

The source is a desktop operations dashboard. It uses a strict twelve-column
grid on charcoal, near-square corners, hairline dividers, and monospaced
numerals. Neon lime appears only on live values and successful state changes;
errors use a narrow vermilion strip rather than filled cards. Labels are terse
nouns. Tables are dense but never zebra-striped. Controls snap into place in
roughly 120 milliseconds with no bounce, blur, gradients, or decorative
illustration. Empty space is used to separate systems, not individual rows.

The downstream artifact will be a new alert-configuration panel in the same
product. Another agent will design it.

Task:

Extract the source's taste into a reusable handoff that lets the next agent
design the alert panel in the same style. Do not design the panel itself.

Your final response is raw working data for analysis, not a user-facing
summary. Report your observations, abstractions, handoff, and result.
