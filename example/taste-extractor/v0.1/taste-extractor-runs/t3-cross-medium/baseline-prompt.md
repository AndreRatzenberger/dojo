# T3 baseline prompt

You are working on the following task. Use whatever approach you think is
right. Do not load an Agent Skill or inspect unrelated files.

Available context:

The source is a collection of hand-thrown breakfast ceramics. Bowls are low and
slightly off-round, with thumb marks left visible beneath a chalky oat glaze.
Each piece has one cobalt brush line that stops before completing the circle.
Objects sit far apart on rough linen. Sets are related but never identical:
three quiet forms, then one unexpectedly tall cup. Nothing is glossy. Repairs
are shown rather than hidden, but decoration is sparse enough that the food
remains the event.

The downstream artifact is a public service-status page. Another agent will
design and implement it.

Task:

Extract the ceramics collection's taste and translate it into a reusable
handoff for the status page. Preserve the character without turning the page
into pottery-themed decoration. Do not design the page itself.

Your final response is raw working data for analysis, not a user-facing
summary. Report your observations, abstractions, translation, handoff, and
result.
