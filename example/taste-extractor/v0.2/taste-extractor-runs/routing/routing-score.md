# Routing score

Expected owners were declared before either judge ran. Two clean proxy judges
received the same isolated eight-skill catalog and fifteen prompts.

| Measure | Result |
|---|---:|
| Candidate-positive prompts | 18/18 correct across two judges |
| Neighbor-owned prompts | 12/12 correct across two judges |
| Total | **30/30** |

This is a targeted routing proxy, not a production-dispatcher benchmark. The
candidate was evaluated as the isolated environment's taste-extraction
capability rather than beside a second same-purpose skill.
