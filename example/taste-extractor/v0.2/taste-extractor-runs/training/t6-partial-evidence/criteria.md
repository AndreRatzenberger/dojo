# T6 criteria

1. The response lists the four accessible frames and explicitly names motion, timing, transitions, and audio as unavailable.
2. It extracts only visually supported observations and principles instead of refusing all useful analysis.
3. It makes no claims about motion, edit rhythm, sound, or temporal arc and labels the profile provisional.
4. It requests the smallest additional evidence needed to resolve temporal and audio unknowns.
5. It provides bounded visual template guidance, confidence notes, and validation checks that do not depend on missing modalities.

Qualitative gate: the provisional handoff should be genuinely useful from the stills while making it impossible to mistake guesses about missing modalities for evidence.
